<h1>🚀 College Events Management System</h1>

<p>
  A modern, scalable, and production-ready College Events Management Web Application built using the MERN Stack.
</p>

<p>
  This platform provides seamless event discovery, ticket booking, seat selection, QR-based tickets, smart recommendations, waitlist management, and organizer analytics — all inside a minimal and responsive interface.
</p>

---

# 🚀 Features

## 🔐 Authentication System
- Ref No + Password Login
- Role-Based Authentication
- Dynamic Dashboard Redirection
- Session Handling


# 🎓 Student Features

- Browse Events
- Smart Recommendations
- Book Tickets
- Seat Selection UI
- Group Booking
- Join Waitlist
- QR Ticket Access
- Transfer Tickets
- Bookmark Events
- Activity Streaks
- Leaderboard System
- Ratings & Reviews


# 🎪 Organizer Features

- Create Events
- Manage Events
- Seat Layout Management
- Booking Analytics
- Waitlist Insights
- Feedback Reports
- Dashboard Statistics


# 👑 Admin Features

- Platform Analytics
- Event Moderation
- User Management
- Reports & Insights


# 🧠 Smart Recommendation System

Recommendations are generated based on:

- User Preferences
- Event Popularity
- Recent Activity
- Previously Viewed Events
- Booked Categories

# 🎫 Ticketing System

- Unique Booking IDs
- QR Code Tickets
- Offline Ticket Access
- Seat-Based Booking
- Ticket Transfer using Ref No


# 🔥 Advanced Features

## 🔄 Waitlist System
- FIFO Queue Logic
- Auto Booking on Cancellation

## 👥 Group Booking
- Book Tickets for Multiple Users
- Invite using Ref Nos

## 🪑 Seat Selection UI
- Interactive Grid Layout
- Real-Time Seat Availability
- Disabled Booked Seats
